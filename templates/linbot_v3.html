채팅하는 동안
실시간 번역 + 발음

GPT-4o가 구어체·인터넷 표현까지 정확하게 번역하고, 히라가나와 한글 발음을 동시에 표기합니다. LINE 기본 번역과는 차원이 다른 품질을 경험하세요.

🔤

GPT-4o 고품질 번역

やばい, それな 등 구어체·인터넷 표현 정확 처리. 맥락을 이해한 자연스러운 번역.

🔊

히라가나 + 한글 발음

今日は → きょうは / 쿄오와. 한자 발음을 두 가지 방식으로 동시 표기합니다.

💬

양방향 자동 전송

한국어 입력 → 일본어 자동 전송 / 일본어 수신 → 한국어 번역. 대화 흐름이 끊기지 않습니다.

🔍

표현 설명 (/explain)

/explain それな → "맞아 그거. 상대 의견에 동의할 때 쓰는 인터넷 표현" 즉시 설명.

퀴즈 자동 생성

방금 나눈 대화의 단어와 문장이 바로 퀴즈가 됩니다. Duolingo처럼 게임하듯 배우고, 포인트를 쌓으면 무료 번역 횟수가 늘어납니다.

🎮

단어 뜻 맞추기

대화 중 나온 단어를 즉시 퀴즈로 출제. 정답 시 +3 번역 포인트.

🔀

문장 배열 퀴즈

단어를 올바른 순서로 배열하는 문장 조합 퀴즈. 일본어 어순을 자연스럽게 익힙니다.

⭐

포인트 → 무료 번역

퀴즈 포인트로 번역 횟수를 충전. 많이 풀수록 더 많이 번역할 수 있습니다.

대화 데이터가
개인 매거진이 되는 과정

단순 요약이 아닙니다. 발화 분석 → LLM 각색 → 시각화 → 승인 → 신뢰도 강화 — 5단계 파이프라인.

STEP 01
🎙️

발화 추출 & 필터링

본인 발화만 수집. 상대방 식별 정보 제외. 금전·성적·정치 등 민감 주제 자동 차단. 여행·음식·문화 등 '좋은 주제' 자동 감지.

→
STEP 02
🧠

세션 종료 후 LLM 분석

대화 세션이 끝나면 GPT-4o가 전체를 분석. 단순 로그 나열이 아니라 경험과 지식을 주제 중심 '글' 형태로 재구성.

→
STEP 03
✍️

문학적 각색 & 스타일 적용

"익선동 한옥 골목의 밤 분위기"처럼 제목 생성. 수필·여행기·잡지 기사 스타일 선택 적용. 장소 정보 자동 연결.

→
STEP 04
✅

사용자 승인 & 편집

AI 초안은 즉시 공개되지 않음. 사용자가 확인 후 공개 / 나만보기 / 삭제를 직접 결정. 콘텐츠 통제권은 항상 사용자에게.

→
STEP 05
📊

신뢰도 태그 & 매칭 강화

저장된 스토리가 #카페 #여행 등 관심 태그 신뢰도(Confidence Score)를 높여 향후 더 정확한 파트너 매칭으로 연결.
