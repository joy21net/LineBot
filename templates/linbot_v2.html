대화 중 실시간
자동 번역 + 발음

LINE / 카카오 채팅방에 봇을 초대하면 모든 메시지가 자동으로 번역됩니다.
기존 LINE 번역의 가장 큰 문제인 구어체·인터넷 표현을 GPT-4o가 정확하게 처리합니다.

🔤

GPT-4o 고품질 번역

やばい, それな 등 구어체와 인터넷 표현도 자연스럽게. LINE 기본 번역과 차원이 다릅니다.

🔊

히라가나 + 한글 발음 표기

今日は忙しい → 「きょうはいそがしい / 쿄오와 이소가시이」. 한자 발음을 동시에 표시합니다.

💬

양방향 자동 전송

내가 한국어로 입력하면 일본어로, 상대가 일본어로 입력하면 한국어로 자동 번역해 채팅방에 함께 표시.

🔍

표현 설명 기능

/explain それな 입력 시 → "맞아 그거. 상대 의견에 동의할 때 쓰는 일본 인터넷 표현"처럼 맥락 설명.

대화에서 나온 단어로
퀴즈 자동 생성

방금 나눈 대화 내용에서 단어·문장 퀴즈를 자동으로 만들어줍니다.
Duolingo처럼 게임하듯 배우고, 포인트를 쌓으면 무료 번역 횟수가 늘어납니다.

🎮

단어 뜻 맞추기

대화에서 나온 단어를 바로 퀴즈로 출제. 선택지 중 정답을 고르면 +3 포인트 적립.

🔀

문장 배열 문제

단어 순서를 맞추는 문장 조합 퀴즈. 일본어 어순을 자연스럽게 익히는 효과.

⭐

포인트 → 무료 번역

퀴즈 정답 포인트로 무료 번역 횟수 충전. 많이 풀수록 더 많이 번역할 수 있습니다.

🏆

그룹 랭킹 경쟁

채팅방 내 랭킹으로 친구들과 경쟁. 이번 주 1위는 누가 될까요?

배우고 싶은 언어의
원어민과 바로 연결

한국인 ↔ 일본인 언어교환 파트너를 매칭해드립니다.
HelloTalk처럼 상대의 언어를 배우면서 내 언어도 가르쳐주는 Win-Win 구조입니다.

🎯

조건 기반 매칭

나이·관심사·학습 목표에 맞는 파트너를 추천. K-팝 팬, 애니메이션, 여행 등 취향으로 필터링.

🔄

언어 교환 구조

내가 일본어를 배우면서, 상대에게는 한국어를 가르쳐줍니다. 번역봇이 대화를 도와줍니다.

💌

LINE으로 바로 채팅

매칭 후 별도 앱 없이 LINE 채팅방으로 바로 연결. 번역봇이 함께해 언어 장벽 없이 대화 가능.

⭐

파트너 리뷰 시스템

대화 후 파트너를 평가. 좋은 파트너와 계속 이어지고, 우수 파트너는 특별 배지를 받습니다.

드라마 대사로
살아있는 일본어 습득

실제 일본 드라마·애니 대사를 학습 콘텐츠로 변환합니다.
드라마 한 장면을 보고, 대사를 익히고, 퀴즈로 확인하면서 자연스럽게 회화 표현을 배웁니다.

🎬

드라마 대사 학습

일본 드라마·애니의 실제 대사를 번역 + 발음 + 상황 설명과 함께 제공. 교과서에 없는 실전 표현.

📖

스토리 기반 흐름

에피소드 → 장면 → 대사 순으로 이야기 흐름 속에서 단어와 문장을 맥락으로 익힙니다.

🎭

장면별 퀴즈

대사를 보고 빈칸 채우기, 문장 완성 등 장면에 맞는 퀴즈로 기억을 강화합니다.

📌

표현 저장 & 복습

마음에 드는 표현을 저장해 나만의 학습 카드로 만들고, 이후 복습 퀴즈가 자동으로 제공됩니다.

이용 방법

LINE에서
4단계로 시작

앱 설치 없이 LINE 봇 추가 한 번으로 바로 시작됩니다.

STEP 01
🔗

봇 친구 추가

QR코드 또는 링크로 LinguaLine 봇을 LINE 친구로 추가합니다.

STEP 02
👥

채팅방에 초대

일본인 친구 채팅방에 봇을 초대. 이후 모든 메시지 자동 번역 시작.

STEP 03
🎮

대화 + 퀴즈

번역·발음 자동 표시, 대화 중 퀴즈 생성, 매칭·스토리 메뉴 활용.

STEP 04
🏆

포인트 적립

퀴즈 정답 시 포인트 적립. 포인트로 무료 번역 횟수를 늘릴 수 있습니다.
